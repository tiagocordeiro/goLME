Como usar

Você pode usar as URLs públicas com o Token. Ex.:

  • /cotacao/{api_key}/
  • /cotacao/{api_key}/json/

ou enviar o Token no cabeçalho da requisição. Ex.:

  • api/
  • api/{date_from}/{date_to} 
'Token': 'XXXxxxXXXxxxXXXxxx'

outros endpoints:

  • chart/
  • chart/{date_from}/{date_to}
  • grafico/
  • grafico/{api_key}/
  • grafico/{date_from}/{date_to}
  • periodo/{date_from}/{date_to}

PlugIn para WordPress

1. Faça o download do arquivo compactado;

1.1 – Você pode subir pelo seu painel do WordPress. Vá em: Plugins -> Adicionar Novo -> Fazer Upload do Plugin (compactado);

1.2 – Você pode subir pelo FTP. Descompacte o plugin e envie a pasta do Plugin Cotação LME para o diretório “Plugins”;

2. Ative o Plugin em seu painel no WordPress

2.1 - Adicione o Token nas configurações do plugin.

2.2 - Adicione o shortcode [lme] em qualquer página.
